# CloudPulse AI

AI-Powered Incident Intelligence Platform for real-time monitoring, anomaly detection, and cybersecurity analytics.

## Features
- Real-time monitoring dashboard
- AI anomaly detection
- Threat severity scoring
- Threat analytics dashboard
- Incident reports
- Live attack source map
- Downloadable reports
- Dark enterprise UI

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- PyDeck

## Project Purpose
CloudPulse AI helps organizations monitor system performance, detect anomalies, analyze threats, and improve operational response using AI-driven insights.
